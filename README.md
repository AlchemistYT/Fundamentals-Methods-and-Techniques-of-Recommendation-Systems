## 前言
在当今数字化时代，信息过载成为人们面临的一大挑战，推荐系统应运而生，成为解决这一问题的关键技术。它在各个领域广泛应用，从电子商务到在线视频，从新闻资讯到社交网络，深刻改变了人们获取信息和服务的方式，提升了用户体验和企业效益。
本书全面系统地介绍推荐系统相关知识，为不同层次读者提供参考与指导。本书独辟蹊径，以围绕用户和项目的不同数据结构表示为主线来审视推荐系统的发展过程，使读者可以对推荐系统的发展和核心问题有个全面、系统地了解和把握。全书共11章，各章紧密相连，逐步深入，并提供相应的示例来指导读者独立开发相应的推荐方法。其中1-5章介绍推荐系统基础概念与经典方法，6-8章介绍基于图数据结构的推荐方法，9-11章介绍基于项目偏序图数据结构的推荐方法。

- 第1章“推荐系统概述”，介绍推荐系统的应用场景，如音乐、视频、教育网站等领域的具体应用实例；梳理推荐方法的分类，包括基于内容、矩阵表示、图表示等方法及其研究进展；阐述推荐系统评价的重要性和多维度视角，涵盖不同利益相关者的关注点；讲解搭建流程，并通过实际代码展示如何构建电影推荐系统，为后续学习奠定基础。
- 第2章“推荐系统评测”，深入探讨评测方法，包括离线实验、用户调查和在线实验，分析它们的优缺点及适用场景；详细介绍各类评价指标，如评分预测、Top - N推荐、基于排序等指标，以及用户满意度、公平性、覆盖范围和商业导向等多方面指标，为准确评估推荐系统性能提供全面依据。
- 第3章“基于内容的推荐方法”，着重讲述项目内容的描述方式，包括数值特征和文本特征的向量表示方法；介绍基于相似度计算和朴素贝叶斯分类器的推荐原理；通过电影推荐实例，展示如何根据用户观看历史推荐相似电影，帮助读者理解基于内容推荐的实际应用。
- 第4章“基于交互矩阵的协同过滤推荐方法”，概述基于交互矩阵的协同过滤方法，阐述其在挖掘用户与项目关系方面的核心思想；详细说明用户行为的矩阵表示，包括交互矩阵的构造和相似度计算方法；深入讲解基于近邻和模型的协同过滤方法，对比两者差异，并通过电影推荐实验，对比不同方法的效果，让读者掌握协同过滤方法的实际应用。
- 第5章“基于项目-类别矩阵的推荐方法”，介绍项目 - 类别矩阵的基础概念，展示其在处理数据稀疏性、增强语义理解和提升可解释性方面的优势；探讨基于特征层次结构和多层表示学习的推荐框架设计，包括框架目标、核心组件和模型学习过程，通过电商平台实例，体现该方法在实际中的应用价值。
- 第6章“基于用户-项目关系图的推荐方法”，提出基于path2rec模型的算法，设计智能游走采样策略以改善数据稀疏问题；阐述模型结构，包括pathGAN和path2vec模块的功能与协作方式；通过在电影推荐数据集上的实验，验证模型在推荐效率上的提升，为读者呈现深度学习在推荐系统中的应用。
- 第7章“基于用户-用户关系图的推荐方法”，定义用户 - 用户关系图，阐述其在社交推荐等方面的应用；描述开发者推荐相关工作，分析现有方法的局限性；设计总体框架，融合多元关系提升推荐准确性；通过GitHub和GitLab数据实验，验证方法在解决数据稀疏和冷启动问题上的有效性，展示用户 - 用户关系在推荐系统中的重要性。
- 第8章“基于知识图谱的推荐方法”，介绍知识图谱在推荐系统中的作用，阐述其如何解决冷启动和数据稀疏问题；详细说明项目 - 实体关系图的构建与应用；深入讲解基于嵌入和基于路径的推荐方法，包括Trans系列图谱嵌入方法和基于元路径、路径推理的推荐方法；通过电影推荐实例，展示基于GNN的知识图谱推荐方法的实现过程，体现知识图谱为推荐系统带来的优势。
- 第9章“基于项目偏序图的会话推荐方法”，分析会话推荐的产生背景，阐述其与传统推荐系统的区别；定义项目 - 项目关系图，说明其在会话推荐中的构建方式和应用；介绍会话推荐系统的概念、任务和特点；详细讲解传统方法和基于神经网络的方法，包括基于模式挖掘、K近邻、马尔可夫链的传统方法，以及基于循环神经网络、卷积神经网络、注意力机制和图神经网络的神经网络方法，通过商品推荐实例，帮助读者理解会话推荐系统的原理和应用。
- 第10章“基于项目偏序图的序列推荐方法”介绍了序列化推荐系统的基本概念，涵盖了基于马尔可夫链、RNN、注意力机制的序列推荐算法，以及项目-项目偏序和物品附加信息的KSR模型等内容。
- 第11章“推荐系统中的实例可靠性问题”探讨了实例可靠性与不可靠数据处理，重点分析了外部干扰因素如何影响序列推荐模型的准确性，并提出通过项目-项目偏序图和项目属性异质图等方法来识别和修正低质量数据，从而提升推荐系统的性能和用户体验。
